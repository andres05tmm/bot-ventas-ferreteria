Fase B · Dashboard Polish

3 direcciones visuales

Cada mockup muestra el tab Hoy con el layout decidido (KPIs pequeños, chart + feed live, sección secundaria abajo). La diferencia entre los 3 es identidad visual: cómo se siente el fondo, las cards y la jerarquía.

A

Aurora Ferretera

Gradient mesh fixed con orbes radiales sutiles · KPIs con tints semánticos · micro-motion en hover. Identidad cálida y suave sin perder densidad.

B

Forja Metálica

Doble borde arquitectónico · accent stripes top en cards · label-caps con tracking wider · franja roja en sidebar activo. Industrial premium tipo Stripe.

C

Bento Card-Led

Accent stripes laterales · sparklines en KPIs · dot-pattern en cards · bento asimétrico abajo. Estilo Linear/Vercel puro, máxima energía en las cards.

Después de elegir

Decime "voy con A" / "con B" / "con C" y la aplico al código (tokens CSS + componentes + layout de Hoy). Si querés combinar elementos de varias (ej. fondo de A + stripes de C), también lo podemos hacer.
