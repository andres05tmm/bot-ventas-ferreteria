## Resumen de cada dirección

### 1 · NCR / Carbón químico
Cuaderno de papel químico amarillo-pálido. Cifras en mono "máquina de escribir", bordes superiores con perforación punteada, sello rotado `-0.5°` para estados. Brand red mate `#B11D0E` como sello de carbón.

### 2 · Specimen / Hoja técnica
Hoja de catálogo industrial McMaster-Carr. Densidad alta, bordes 0.5px gris frío, leader-lines con callouts técnicos (`n=42`, `±σ`), tick marks de regla CAD en márgenes. Negro sobre blanco puro, brand red reservado solo para alertas críticas.

### 3 · Ledger / Libro contable
Papel ledger envejecido con líneas guía azul tinta cada 28px. Tabla doble columna Debe/Haber, anotaciones marginales manuscritas (Caveat), sellos rotados "LIQUIDADO / ORIGINAL / SELLO DE AUDITORÍA". Verde `#2F6B3F` para ingresos, rojo brand para egresos.

### 4 · Red-Drenched / 60% brand
Identidad committed: sidebar entera en rojo oscuro `#7A1206`, hero mega-card en brand red `#C8200E` con cifra blanca Inter 64px. KPIs blancos sobre crema. Acentos dorados `#D4A95C`. Sin gradientes.
